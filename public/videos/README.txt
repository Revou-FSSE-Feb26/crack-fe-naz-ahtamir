# K3 Video Folder

Tempatkan file video K3 (Keselamatan dan Kesehatan Kerja) di sini.

## Rekomendasi:
- Format: MP4 (H.264 codec)
- Resolusi: 1920x1080 (Full HD)
- Durasi: 30-60 detik
- Ukuran: < 50MB

## Contoh nama file:
- k3-implementation.mp4
- k3-cinematic.mp4
- nickel-smelter-safety.mp4

## Cara mengganti di index.html:
Ganti source video di baris 147:
```html
<source src="videos/your-video-file.mp4" type="video/mp4" />
```
